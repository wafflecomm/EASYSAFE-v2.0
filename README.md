# 이지세이프 (EASY SAFE) 🛡️

> **"사업주는 EASY하게, 근로자는 SAFE하게"**  
> 스마트 안전보건관리 시스템 - 중대재해처벌법/EHS 플랫폼

---

## 🌟 프로젝트 개요
**이지세이프(EASY SAFE)**는 복잡한 안전보건관리 업무를 디지털로 전환(DX)하여 사업주와 근로자 모두에게 편리하고 안전한 작업 환경을 제공하는 스마트 솔루션입니다. **중대재해처벌법**, **산업안전보건법**, **건설기술진흥법** 등 강화된 법적 의무를 모바일 앱 하나로 100% 충족할 수 있도록 설계되었습니다.

## ✨ 8대 핵심 기능 (Smart Tools)
현장의 목소리를 반영한 직관적이고 강력한 기능을 제공합니다.

1.  **📱 모바일 점검**: 현장에서 즉시 사진 촬영 및 위험 요인 등록/검수.
2.  **🎙️ 보이스 입력**: 타자가 어려운 현장직을 위한 혁신적인 음성 텍스트 변환(STT) 입력.
3.  **👓 시니어 맞춤 UI**: 큰 글씨와 명확한 아이콘으로 누구나 돋보기 없이 사용 가능.
4.  **🕹️ 컨디션 체크**: 미니게임을 통한 작업자 건강 상태 및 집중도 매일 확인.
5.  **📜 보고서 자동 생성**: 입력된 데이터로 위험성 평가 및 TBM 회의록 자동 문서화.
6.  **✍️ 전자서명**: 법적 효력을 갖춘 모바일 서명으로 페이퍼리스(Paperless) 실현.
7.  **💬 카카오 알림톡**: 주요 안전 업무 및 조치 사항 실시간 공유 및 누락 방지.
8.  **🏷️ TAG 공유**: `#` 태그를 활용한 부서 간 즉시 정보 공유 및 작업 관리 연동.

## 👨‍⚖️ 법적 의무 자동 준수 (Compliance)
이지세이프를 사용하는 것만으로도 다음 법령의 의무 사항이 자연스럽게 이행됩니다.
- **산업안전보건법**: 유해·위험 요인 정기 점검, TBM 안전교육 실시 인정.
- **중대재해처벌법**: 경영책임자의 안전보건 확보 의무 이행 증명.
- **건설기술진흥법**: 공사 현장 안전 관리 시스템 구축 및 운영.

## 🎨 디자인 및 기술 스택
**Supanova 프리미엄 스타일**을 적용하여 모바일 앱과 같은 트렌디한 사용자 경험을 제공합니다.

-   **Frontend**: HTML5, Vanilla JavaScript.
-   **Styling**: [Tailwind CSS](https://tailwindcss.com/) (CDN).
-   **Design**: 글래스모피즘(Glassmorphism), 앰비언트 글로우(Ambient Glow), 페럴랙스(Parallax) 애니메이션.
-   **Typography**: [Pretendard](https://github.com/orioncactus/pretendard) (가독성 높은 한국어 표준 글꼴).
-   **Icons**: [Iconify](https://icon-sets.iconify.design/) (Solar & MDI sets).

## 🚀 시작하기
이지세이프의 랜딩페이지를 로컬 환경에서 확인하는 방법입니다.

1.  저장소를 클론합니다.
2.  루트 디렉토리의 `index.html` 파일을 최신 웹 브라우저(Chrome, Edge 등)로 엽니다.
3.  별도의 서버 설정 없이 즉시 프리미엄 디자인과 애니메이션을 확인하실 수 있습니다.
